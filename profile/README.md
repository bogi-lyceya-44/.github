## Что используем?

### Бэкенд

- gRPC, REST
- Swagger - API документация

- Zap - логгинг
- Jaeger - трейсинг запросов
- Prometheus & Grafana - сбор и визуализация метрик
- pprof - профилирование

- goose - миграции
- mockery - моки

- Kafka - брокер сообщений
- PostgreSQL - основная БД

- Docker - контейнеризация
